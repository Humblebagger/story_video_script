# 多题材稳定性测试报告（2026-07-11）

测试对象：`prompts/novel_to_storyboard.md`（测试时为初版；本报告发现的问题已修入现行版本）
方法：三段原创测试文本（各 ~200 字，埋入针对性压力点）→ 独立 LLM 代理按 prompt 转换 → 三层验证（lint 结构+引用+覆盖率 / 原文保真度逐字比对 / Seedance 适配器渲染）。
测试材料与输出见 `tests/genre_stability/`。

## 结果总览：3/3 全部通过三层验证

| 题材 | 压力点 | 句子 | 镜头 | inferred占比 | 三层验证 |
|---|---|---|---|---|---|
| 言情（现代校园） | 双人对白快速交替、心理闪回 | 11 | 11 | 0.18 | PASS |
| 玄幻（修仙战斗） | 招式特效、**战斗中受伤换装** | 11 | 11 | 0.09 | PASS |
| 古典白话（武侠古风） | 文言语汇、**主语省略、无标记引语** | 13 | 14 | 0.07 | PASS |

亮点验证：
- **换装状态机**（玄幻）：受伤发生在 u0006，SH006 起 outfit 精确切换为 `wounded_left_shoulder`，此前镜头均为 `default`
- **文言指称消解**（古典）："之""李郎""他"正确归入同一角色 aliases；纯引语句"粮草，为何迟了七日？"跨句推断出说话人
- **群像即兴处理**（古典）："满座朱紫"自发建为群像卡 C03（当时规则未覆盖，处理合理，已固化进规则）

## 发现的规则缺陷（已修复）

**P0 · 规则内部矛盾——对白句非引语部分 vs 旁白重复朗读**
言情和古典独立撞上且处理相反：言情把整句进旁白（成片会旁白+角色重复念台词），古典把整句踢出旁白（丢失"太傅举杯遥指之，笑曰"的动作信息）。
修复：规则 7 新增唯一允许的删减——对白句进旁白时剔除引号内台词、保留非引语片段。

**P1 · 追述/回忆句的 derivation 归类无裁决**（玄幻、古典都撞上）
修复：规则 2 明确——判定依据是画面元素是否来自原文字面，与呈现手法无关；原文明确描写的追述做成闪回仍算 explicit。

**P1 · 镜头层禁外貌词与原文动作冲突**（言情"裙摆"、古典"白衣"）
修复：规则 5 明确——禁的是设定性描述（款式/颜色/五官），动作必然涉及的衣物部位可用中性指称。

**P2 · 五项小缺口**：群像建卡方式、复合句 kind 取主导、纯台词镜头省略 narration、time_of_day 无依据时弱推断、inferred_shot_ratio 舍入方式——均已写入新增的「边界裁决细则」一节。

## 遗留观察（暂不动）

- 古文超短句（"座中皆笑。"4 字）导致 3 秒短镜头偏多——节奏问题属 pacing_optimized 模式的范畴，忠实模式不处理
- 单句闪回也要求建完整场景卡（言情 S02）——成本略高但换来一致性，维持现状
- schema 仍未收紧 `additionalProperties`，字段漂移靠 prompt 骨架约束——列在 README roadmap

## 追加：selective 旁白模式回归测试（同日）

设计变更：旁白与忠实性解耦——句子可由旁白/台词/画面三种方式承载，新增 `narration.mode = selective`（能拍出来的不念），覆盖率校验口径随模式切换；`original_text` 模式行为不变（5 份旧输出回归全部 PASS）。

用修订版 prompt + selective 模式重跑古典篇：13 句 → 15 镜头，承载分布为旁白 4 / 台词 3 / 纯画面 6，三层验证 PASS。上一轮修订的规则全部按预期执行：追述句闪回正确判为 explicit（inferred 占比从 0.07 降到 0）、群像卡按简化规则建卡。本轮又暴露并修复 3 处小模糊点：文学性条款限定为点题句、群像卡 outfits 可省、场景卡建卡门槛（取过景一律建卡）。

## 结论

转换 prompt 在四种差异极大的定制文风 + 长章节分批 + 衣物高密度场景 + **真实公版全文**上均产出结构合法、原文逐字保真、引用自洽的分镜 JSON，核心机制（溯源锚点、推导标记、资产引用制、换装状态、资产库续传、匿名卡揭示、BGM 曲线）稳定。主要风险不在结构层而在规则边界的裁决一致性——四轮共暴露 51 处，均已修入 prompt、schema（v0.2 严格模式 / v0.3 music / v0.4 道具状态机·生物卡·旁白顺序）、工具或裁决细则，无遗留项。《药》归档已同步改造为 v0.4 特性的实战验证件（人血馒头五态精确到镜头、乌鸦 A01 生物卡、五少爷镜头 narration.order）。真实文本的实测印证了此前判断：定制压力文测不出切分层问题（不可见字符、章节标记、底本讹字），真实底本是必要的回归材料。
